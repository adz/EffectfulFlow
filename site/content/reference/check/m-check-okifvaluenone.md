Returns success when the value option is <code>ValueNone</code>.

## Parameters

- `opt`: <code><span>'a&#32;voption</span></code>
  The value option to check.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> that succeeds if the value option is <a href="https://fsharp.github.io/fsharp-core-docs/reference/fsharp-core-fsharpvalueoption-1-valuenone">ValueNone</a>.
