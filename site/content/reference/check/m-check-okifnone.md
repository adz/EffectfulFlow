Returns success when the option is <code>None</code>.

## Parameters

- `opt`: <code><span>'a&#32;option</span></code>
  The option to check.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> that succeeds if the option is <a href="https://fsharp.github.io/fsharp-core-docs/reference/fsharp-core-fsharpoption-1-none">None</a>.
