Returns success when the values are not equal.

## Parameters

- `expected`: <code>'a</code>
  The expected value.
- `actual`: <code>'a</code>
  The actual value.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> that succeeds if the values differ.
