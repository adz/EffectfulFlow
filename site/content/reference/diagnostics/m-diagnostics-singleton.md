Creates a diagnostics graph containing exactly one error at the root.

## Parameters

- `error`: <code>'error</code>
  The application error to store at the root.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-diagnostics-1.html">Diagnostics</a> with a single error.
