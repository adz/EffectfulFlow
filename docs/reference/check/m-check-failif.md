Returns success when the condition is false.

## Parameters

- `cond`: <code>bool</code>
  The boolean condition to check.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> that succeeds if <span class="fsdocs-param-name">cond</span> is false.
