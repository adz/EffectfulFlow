Returns the value when the value option is <code>ValueSome</code>.

## Parameters

- `opt`: <code><span>'a&#32;voption</span></code>
  The <a href="https://fsharp.github.io/fsharp-core-docs/reference/fsharp-core-fsharpvalueoption-1">voption</a> to check.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> containing the value if present.
