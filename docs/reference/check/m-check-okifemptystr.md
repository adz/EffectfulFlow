Returns success when the string is null or empty.

## Parameters

- `str`: <code>string</code>
  The string to check.

## Returns

A <a href="https://adz.github.io/FsFlow/reference/FsFlow/fsflow-check-1.html">Check</a> that succeeds if the string is null or empty.
